## ­ЪЊІ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ
[рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ](https://www.kaggle.com/competitions/walmart-recruiting-store-sales-forecasting) рЃљрЃарЃўрЃА Walmart-рЃўрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃюрЃДрЃЮрЃцрЃўрЃџрЃћрЃЉрЃўрЃА рЃДрЃЮрЃЋрЃћрЃџрЃЎрЃЋрЃўрЃарЃћрЃБрЃџрЃў рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃљ 2010-2012 рЃгрЃџрЃћрЃЉрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃЏрЃћрЃбрЃарЃўрЃЎрЃљрЃљ `Weighted Mean Absolute Error`, рЃАрЃљрЃЊрЃљрЃф рЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃЎрЃЋрЃўрЃарЃћрЃЉрЃўрЃА рЃгрЃЮрЃюрЃљ 5-рЃ»рЃћрЃа рЃБрЃцрЃарЃЮ рЃЏрЃљрЃдрЃљрЃџрЃўрЃљ, рЃЋрЃўрЃЊрЃарЃћ рЃљрЃарЃљрЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃЎрЃЋрЃўрЃарЃћрЃЉрЃўрЃА. рЃћрЃА рЃљрЃАрЃљрЃ«рЃљрЃЋрЃА рЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃЎрЃЋрЃўрЃарЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃќрЃБрЃАрЃбрЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА рЃЉрЃўрЃќрЃюрЃћрЃАрЃўрЃАрЃЌрЃЋрЃўрЃА.

## ­ЪЏа№ИЈ рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ

рЃљрЃЏ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЏрЃљрЃюрЃЦрЃљрЃюрЃБрЃарЃў рЃАрЃгрЃљрЃЋрЃџрЃћрЃЉрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ Pipeline. рЃЋрЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃўрЃЌ:

1. **Exploratory Data Analysis -** рЃарЃЮрЃЏ рЃерЃћрЃњрЃЋрЃћрЃАрЃгрЃљрЃЋрЃџрЃљ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЎрЃљрЃюрЃЮрЃюрЃќрЃЮрЃЏрЃўрЃћрЃарЃћрЃЉрЃћрЃЉрЃў, рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃЮрЃЉрЃљ рЃЊрЃљ рЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃЋрЃџрЃћрЃюрЃљ.
2. **Feature Engineering -** *time series* рЃбрЃћрЃЦрЃюрЃўрЃЎрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃЋрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃљрЃ«рЃљрЃџрЃў feature-рЃћрЃЉрЃў, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ lag features рЃЊрЃљ рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃЮрЃЉрЃўрЃА рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃћрЃЉрЃў.
3. **рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ -** рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЎрЃџрЃљрЃАрЃўрЃЎрЃБрЃарЃў рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃБрЃарЃў рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў, tree-based рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў рЃЊрЃљ deep learning рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў.
4. **MLflow рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ -** рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃбрЃарЃћрЃЎрЃўрЃюрЃњрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЌ MLflow.
5. **Pipeline-рЃўрЃА рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ -** Pipeline-рЃћрЃЉрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃћрЃю рЃЎрЃЮрЃюрЃАрЃўрЃАрЃбрЃћрЃюрЃбрЃБрЃа рЃърЃарЃћрЃърЃарЃЮрЃфрЃћрЃАрЃўрЃюрЃњрЃА, рЃбрЃарЃћрЃюрЃўрЃюрЃњрЃА рЃЊрЃљ рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃљрЃА рЃАрЃљрЃбрЃћрЃАрЃбрЃЮ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃќрЃћ.

## ­ЪћД рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ
- рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃўрЃюрЃЊрЃћрЃЦрЃАрЃўрЃарЃЊрЃћрЃЉрЃљ `Date`, `Store` рЃЊрЃљ `Dept` рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃЌ.
- рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌ рЃАрЃЋрЃћрЃбрЃћрЃЉрЃерЃў `NaN` рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃўрЃЋрЃАрЃћрЃЉрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃўрЃЌ, рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃБрЃџрЃерЃў рЃЎрЃў Рћђ рЃЏрЃЮрЃЊрЃўрЃЌ.
- рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃБрЃџрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў рЃњрЃљрЃЊрЃљрЃЊрЃўрЃА рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌрЃерЃў `OneHotEncoder`-рЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃЌ.
- рЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃЮрЃЉрЃўрЃА feature-рЃћрЃЉрЃў, рЃљрЃАрЃћрЃЋрЃћ lag feature-рЃћрЃЉрЃў.
  - рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў рЃерЃћрЃўрЃарЃЕрЃљ EDA-рЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃЌ.
  - рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў рЃфрЃўрЃЎрЃџрЃБрЃарЃў рЃърЃћрЃарЃўрЃЮрЃЊрЃћрЃЉрЃў рЃўрЃДрЃЮ 13 рЃЊрЃљ 23 рЃЎрЃЋрЃўрЃарЃўрЃљрЃюрЃў.
  - рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃФрЃџрЃўрЃћрЃарЃў рЃљрЃЋрЃбрЃЮрЃЎрЃЮрЃарЃћрЃџрЃљрЃфрЃўрЃљ рЃўрЃДрЃЮ 1 рЃЊрЃљ 52 рЃЎрЃЋрЃўрЃарЃўрЃљрЃю рЃЕрЃљрЃЏрЃЮрЃарЃЕрЃћрЃюрЃћрЃЉрЃќрЃћ.
- рЃўрЃерЃџрЃћрЃЉрЃљ рЃљрЃарЃљрЃАрЃљрЃГрЃўрЃарЃЮ рЃАрЃЋрЃћрЃбрЃћрЃЉрЃў.
- рЃЉрЃЮрЃџрЃЮрЃА, рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃњрЃљрЃЊрЃљрЃћрЃфрЃћрЃЏрЃљ рЃЏрЃЮрЃЊрЃћрЃџрЃА рЃбрЃарЃћрЃюрЃўрЃюрЃњрЃўрЃАрЃЌрЃЋрЃўрЃА рЃљрЃю рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

## ­ЪЊі MLflow рЃбрЃарЃћрЃЎрЃўрЃюрЃњрЃў

рЃДрЃЋрЃћрЃџрЃљ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў рЃЊрЃљрЃџрЃЮрЃњрЃўрЃџрЃўрЃљ [MLflow](https://dagshub.com/gnada22/ml_final_project.mlflow/#)-рЃќрЃћ:
- рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃЦрЃБрЃџрЃћрЃЉрЃў (`MAE`, `WMAE`)
- рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў feature-рЃћрЃЉрЃў
- рЃЌрЃЋрЃўрЃЌрЃЮрЃю рЃЏрЃЮрЃЊрЃћрЃџрЃў
